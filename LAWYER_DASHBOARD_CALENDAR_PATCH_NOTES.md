# Lawyer Dashboard Calendar Patch

تغییرات صفحه `/dashboard/lawyer`:

- صفحه از حالت تب‌محور خارج شد و همه چیز داخل «نمای کلی» قرار گرفت.
- دکمه/تب «رزروها» حذف شد و رزروها در نمای کلی نمایش داده شدند.
- دکمه/تب «پروفایل» حذف شد و پروفایل در نمای کلی نمایش داده شد.
- لینک «مشاهده سایت» حذف شد.
- مدیریت رزروها به تقویم ساده ۱۴ روزه تبدیل شد.
- آمارهای کل رزروها، در انتظار، تایید شده و انجام شده مثل پنل کاربر طراحی شد.
- دکمه خروج مثل پنل کاربر به پایین صفحه منتقل شد و به صفحه اصلی `/` ریدایرکت می‌شود.
- صفحه برای موبایل ریسپانسیو شد.
