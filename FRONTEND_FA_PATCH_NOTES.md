# Lexara Persian Responsive UI Patch

این نسخه روی فرانت‌اند پروژه اعمال شده است:

- فارسی‌سازی کامل صفحات اصلی فرانت شامل خانه، ورود، ثبت‌نام، لیست وکلا، جزئیات وکیل، پنل کاربر و پنل وکیل.
- فعال‌سازی RTL از طریق `html lang="fa" dir="rtl"`.
- بازطراحی `globals.css` با فونت فارسی Vazirmatn، گریدهای واکنش‌گرا، کارت‌های glass، انیمیشن‌های نرم و حالت موبایل.
- ریسپانسیو شدن navbar، فرم‌ها، کارت‌ها، داشبوردها، لیست‌ها و مودال رزرو برای موبایل/تبلت/دسکتاپ.
- تفکیک مسیر حساب کاربر و وکیل:
  - کاربر: `/dashboard/customer`
  - وکیل: `/dashboard/lawyer`
- ورود و ثبت‌نام با هدایت خودکار بر اساس نقش برگشتی از API.
- امن‌تر شدن ذخیره کوکی‌ها در فرانت با `sameSite: 'lax'` و `secure` در حالت production.
- حفظ APIهای فعلی پروژه بدون تغییر در بک‌اند.

برای اجرا:

```bash
cd frontend
npm install
npm run dev
```

اگر backend روی آدرس دیگری است، مقدار `NEXT_PUBLIC_API_URL` را در `.env.local` تنظیم کن.
